前端代码规范

发表于 2017-05-19
  1. 尽量避免用var,多用let和const
  2. 定义对象用对象字面量
    eg: const obj = {}
  3. 定义对象的属性和方法，尽量用速写法

  4. 少用Object.assign()

  5. 用… to copy arrays

  6. 多行数组的定义方式

  7. 尽量用解构赋值符

  8. 定义string尽量用’’

  9. 箭头函数

  10. use === and !== over == and i=

  11. switch…case…

  12. 取值判断
